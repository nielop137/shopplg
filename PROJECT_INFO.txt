MAIN FEATURES
═══════════════════════════════════════════════════════════════════════════

✓ Three Shop Types:
  • Vault Shop      - Economy money (Vault integration)
  • Time Shop       - Time points (earned by playing)
  • Wallet Shop     - Virtual currency (plugin wallet)

✓ Time Points System:
  • Automatic point awards
  • AFK detection
  • Configurable intervals
  • Playtime tracking

✓ Virtual Wallet:
  • Independent from Vault
  • Admin management commands
  • PlaceholderAPI support

✓ Easy Item Management:
  • GUI-based item addition
  • No YAML editing required
  • Full NBT preservation
  • Custom item support (MMOItems, Oraxen)

✓ Modern GUI:
  • Animated interfaces
  • Purchase confirmation
  • Stack buying options
  • Pagination support

✓ Flexible Storage:
  • SQLite (default)
  • JSON (optional)
  • Auto-migration

✓ Full Integration:
  • Vault (required)
  • PlaceholderAPI (optional)
  • Custom items plugins

COMMANDS
═══════════════════════════════════════════════════════════════════════════

Admin:
├── /nieloshop reload       - Reload configuration
├── /nieloshop save         - Save all data
└── /nieloshop add <shop>   - Add item to shop

Player:
├── /shop <vault|time|wallet>  - Open shop
├── /portfel                   - Check wallet
├── /portfel nadaj <p> <a>     - Give wallet currency (admin)
└── /czas                      - Check playtime & points

PERMISSIONS
═══════════════════════════════════════════════════════════════════════════

nieloshop.use              - Open shops (default: true)
nieloshop.admin            - Admin commands (default: op)
nieloshop.portfel.*        - Wallet management (default: op)
nieloshop.reload           - Reload config (default: op)

PLACEHOLDERS
═══════════════════════════════════════════════════════════════════════════

%nieloshop_wallet%         - Player's wallet balance
%nieloshop_time_points%    - Player's time points
%nieloshop_playtime%       - Player's total playtime

INSTALLATION
═══════════════════════════════════════════════════════════════════════════

1. Install Vault + Economy plugin
2. (Optional) Install PlaceholderAPI
3. Copy NieloShop-1.0.0.jar to plugins/
4. Start server
5. Configure config.yml and messages.yml
6. Add items: /nieloshop add <vault|time|wallet>
7. Done!
